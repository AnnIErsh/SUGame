### Table of contents

## General info
This project is a simple attention game to kill time
	
## Technologies
Project is created with:
* SwiftUI
* SpriteKit
* MVVM
* Sources: EmojiHub free Public Api
	
## Flow
Try to catch emoticons in the right category\
Avoid collisions with wrong objects\
Level up your score
