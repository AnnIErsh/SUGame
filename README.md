## General info
This project is a simple attention game to kill time
	
## Technologies
Project is created with:
* SwiftUI
* SpriteKit
* MVVM/SOA/DI
* Sources: EmojiHub free Public Api
	
## Flow
Try to catch emojis in the right category\
Avoid collisions with wrong objects\
Level up your score
